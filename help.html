Loading Files

To load files:

  1. Click on "Choose Files" button, to open teh file browser
  2. Select the files you want to open
  3. Click on the "Open" button to close the dialog
  4. Click confirm to load the files

Managing Plotted objects

Change label:to rename an object, click on the icon, edit the label and select "Update Label" button

Change color:to change the color of an object, click on the current color and select a color from the list of options

Hide table:to hide a table, click on the icon, and that path will be hidden

Show table:to show a hiddentable, click on the icon, and that path will be shown

Change Time

To change the start and end times to be plotted, drag the handles of the sliders below the graph
