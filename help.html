Loading Files

To load files:

  1. Click on "Load Files" in menu bar, to open the file browser
  2. Select the files you want to open
  3. Click on the "Open" button to close the dialog

Tables

There are three tables shown, only the first will be populated initially (sorted by Reads).

To populate the other tables, select a DSN (row) on the "VSAM Files aggregated by DSN". This will populate the other two tables, providing more details of the DNS selected. One is an aggregated view of that DSN by JOB. THe second are the raw records for that job.

It is possible to further filter the raw records by selecting a JOB in the "Aggregated by Job table".

Sort by Column:to sort by a particular colum, click on the header to initially sort smallest to largest, then click again to sort largest to smallest

Column Visibility:to show or hide columns in the table

  1. Click on "Column visibility" button, to open list of columns
  2. Select the columns you wish to be visible
  3. Click outside the list of columns

Excel Download:to download an excel version of the table, click on the "Excel" button below the title of the table

Filters

There are a number of filters available to filter the results. if a filter is altered, the "Apply Filters" button must be selected to recaculate the DSN table. Note, other tables will only be repopulated once a row on the updated first table is selected

Maximum File Size:this filter eliminates any record that has a files size (in GB) that is shows the VSAM file is larger than the limit show. Defaults to 2

Regex include:this filter eliminates any record where the DSN doesn't match the Regex expression. Defaults to ^[A-Z0-9]

Regex include:this filter eliminates any record where the DSN matches the Regex expression. Defaults to empty

Job Regex include:this filter eliminates any record where the doesn't match the Regex expression. Defaults to empty

Start Time and Finish Time:these two values are used to select a period of the day that is to be used, records outside the range will be eliminated when calculating the result. If the Finish Time is less than the Start Time, the period selected will be the period including midnight.
